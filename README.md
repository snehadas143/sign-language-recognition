# Sign Language Recognition System

A machine learning based sign language recognition system using MediaPipe and Random Forest.

## Features
- Hand detection using MediaPipe
- Landmark extraction
- Dataset collection
- Machine Learning model training
- Real-time sign prediction (A, B, C, D, E)

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Scikit-Learn
- Pandas

## Accuracy
Approximately 89%
